GARANTIA

UMLUseCase
Projeto_Mold :: model_ENTREGA1 :: GARANTIA

Description

Pré-condições O solicitante deverá ter realizado a seleção de uma data e horário definidas como disponíveis para agendamento

Pós-condições A oficina credenciada verifica a disponibilidade da data agendada e retorna se irá realizar o serviço desejado pelo solicitante

Fluxo Principal

1 - Cliente acessa a lista de pedidos

2 - Sistema exibe a lista de pedidos

3 - Cliente acessa o pedido que deseja solicitar garantia

4 - Sistema exibe os detalhes do pedido

5 - Cliente preenche o formulário

6 - Sistema analisa a solicitação

7 - Sistema envia ao cliente o código de postagem

8 - Cliente posta o produto no correio

9 - Oficina analisa o produto

10 - Sistema devolve o dinheiro para o ator

11 - Cliente recebe o dinheiro de volta

Fluxo alternativo:

1 - Recebe uma nova solicitação de agendamento, com as especificações de serviço e data desejada

2 - Retorna para o solicitante a recusa do agendamento

3 - Decide seguir o agendamento com uma nova data ou cancelar o pedido

Fluxo de exceção:

1 - Realiza o cancelamento do serviço anteriormente confirmado, devido algum fator interno

2 - notifica o solicitante do motivo do cancelamento e sugere uma próxima data disponível

3- Escolhe seguir com uma data alternativa ou cancelar a solicitação

4 - Caso o solicitante decida prosseguir com uma nova data, será realizada a confirmação do agendamento
